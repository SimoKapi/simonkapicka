PlantIdentifier

A program which takes an input in the form of a list of plant names, in latin or english, and produces a phylogenetic tree of those plants, categorizing them into their respective Kingdom, Phylum, Class, Order, Family, Genus and Species.

This program was created as a result of a homework assignment in my sophomore Biology class, in which I was supposed to obtain photographs of 25 or more different species of plants and create a phylogenetic tree dividing them all into their Kingdom, Phylum, Class, Order, Family, Genus and Species. This is where I decided to create a simple program to automate the entire process, as not only would I have had to manually search up these levels of classification for each plant individually, I would also have had to graphically design and create the phylogenetic tree itself, which would have ended up taking a long time.
